# 默认视频 v5 与封面质量门

## 背景

默认视频 v5 的产品定义是“作品封面视频”：使用一张高质量 16:9 专辑封面搭配原曲音频，通过 FFmpeg 快速封装为 MP4。

该方案不再把 Remotion 全曲逐帧渲染作为默认路径，也不做无可靠时间轴的强同步歌词。默认视频的审美质量主要由封面决定，因此封面生成必须成为主质量门。

## 默认视频 v5 决策

- 默认视频使用静态专辑封面 + 音频。
- 封面固定 16:9，推荐 `1920x1080` 或 `2048x1152`。
- 视频合成使用 FFmpeg：H.264 video + AAC audio，`yuv420p`，`faststart`。
- 不裁切封面主体；如源图不是 16:9，必须走 contain / pad 或重新生成封面，不允许强行裁没主体。
- 默认不显示歌词卡、不显示逐句字幕、不显示波形动画。
- 实验/高级 Visualizer、歌词 MV、弱歌词卡都不再作为当前产品路径；默认视频只保留快速封面视频。

## 封面质量原则

封面不是普通插图，而是用户作品最终会被社区展示、播放和分享的主视觉。默认视频没有复杂动效兜底，因此封面必须满足正式音乐作品级别。

封面必须：

- 与歌曲标题、摘要、歌词情绪和燕云气质一致。
- 有清晰视觉焦点，适合 16:9 视频首帧。
- 构图高级、克制、有电影感或正式专辑感。
- 画面可在移动端小尺寸下识别。
- 不依赖后期裁切才能成立。

封面不得：

- 出现假歌手名、假演唱者、假制作人、假厂牌、假版权、小字署名。
- 出现乱码、伪汉字、不可读英文、随机水印、UI、排行榜、二维码。
- 变成廉价古风网游海报、活动海报、PPT 背景或游戏截图界面。
- 编造官方未确认角色、场景、阵营结论或剧情关系。
- 使用现实明星、真实歌手肖像或疑似真人肖像。

## 文字策略

当前策略允许 Image2 直接生成带标题的封面，但只允许以下文字：

- 歌曲主标题。
- 可选品牌短字样：`燕云乐坊`。

禁止 Image2 生成：

- `演唱`、`作词`、`作曲`、`制作人`、`出品`、`版权`、`label`、`feat`、人名、艺名、厂牌名。
- 密集小字、长副标题、宣传语、口号和无法验证的署名。

如果连续样本中 Image2 的标题文字不稳定，必须切换为“无字封面 + 后端标准字体叠加歌名”的工程方案。该方案虽然少一点生成感，但可控性更高，适合商业交付。

## 生成前质量门

`CoverPromptAgent` 输出封面 prompt 后，必须先做文本质量门，再调用 Image2。

检查项：

- 是否固定 16:9 输出尺寸。
- 是否明确这是正式音乐作品封面，而不是宣传海报。
- 是否只允许歌名和可选 `燕云乐坊`。
- 是否禁止假署名、假版权、小字、UI、水印和乱码。
- 是否和歌词情绪、音乐方向、燕云气质一致。
- 是否避免复杂正脸、明星脸、具体官方角色和未确认剧情。

失败处理：

- Prompt 含假署名、仿真人、其他 IP 或版权风险：`BLOCK`。
- Prompt 质量普通、构图不清、文字约束不清：`REWRITE`。
- Prompt 合格：进入 Image2。

## 生成后质量门

生成后必须先做技术校验：

- 图片可解码。
- 宽高为 16:9，或可无裁切安全适配到 16:9。
- 最小尺寸不低于 `1920x1080`。
- 文件大小合理。
- MIME 类型正确。
- 对象存储写入成功。

生成后视觉质量门分两层：

1. 自动可判定项：
   - 检查尺寸、解码、文件大小、黑屏/纯色/低信息量。
   - 后续可接 OCR，发现 `演唱`、人名、厂牌、版权、小字密集、乱码时自动拒绝。

2. 人工或多模态审核项：
   - 是否像正式专辑封面。
   - 是否高级克制。
   - 是否主体清晰。
   - 是否燕云气质足够。
   - 是否有低质 AI 痕迹。

在未接入稳定视觉审核前，真实用户测试阶段允许人工抽查；正式交付前必须补自动检测或公司侧审核接入策略。

## 默认视频验收

默认视频 v5 的每个发布包至少满足：

- audio 存在，可播放。
- cover 存在，可解码，16:9。
- video 存在，可播放。
- MP4 包含 H.264 video + AAC audio。
- 视频时长与音频时长基本一致，误差建议小于 0.5 秒。
- 单首 3-4 分钟歌曲，本地/普通服务器合成目标为 10-30 秒。
- 视频首帧不裁切封面主体。
- 没有无时间轴字幕、错位歌词卡或播放器 UI。

样片作品：`我自为舟`

- `work_id`: `9b0e4c9c-e3d2-4866-bb4f-97a43004785e`
- 输出：`build/exports/video-v5-default-wo-zi-wei-zhou/9b0e4c9c-e3d2-4866-bb4f-97a43004785e-v5-default-trimmed.mp4`
- 封面预处理：约 `0.232s`
- MP4 合成：约 `12.594s`
- 总耗时：约 `12.826s`
- 输出：H.264 1920x1080 30fps + AAC 双声道 48kHz
- 文件大小：约 `15.73MB`

当前样片证明默认视频合成路径可达到十几秒级；但封面原图包含不应出现的“演唱/人名”，因此封面质量门必须先修，不能直接把当前封面策略视为商用品质。
