# ADR 0001: 用户侧 Web 范围

日期：2026-06-05

## 状态

已决策

## 背景

燕云 AI 作曲平台需要让玩家完成故事输入、歌词确认、生成状态查看、成品预览和作品列表管理。用户侧体验既要适配移动端 H5/WebView，也要支持 PC Web，便于深度创作者粘贴长歌词、运营验收和公司开发联调。

平台不负责社区发布、分享、点赞、评论、推荐流等外部能力，这些由公司既有系统承接。

## 决策

- 用户侧由本项目提供 `apps/web`。
- `apps/web` 使用 React + Vite + TypeScript。
- 设计原则为移动端优先，同时兼容 PC Web。
- `apps/web` 只调用 `music-api`，不直接访问数据库、对象存储、Provider 或公司真实系统。
- 第 1 批仓库初始化创建 `apps/web` scaffold，但不实现完整业务页面。

## 影响

- PRD v0.3 和技术方案 v0.2 均以移动端优先、兼容 PC Web 为用户侧范围。
- OpenAPI v0.1 必须覆盖前端需要的作品状态、生成阶段、权益提示、失败动作和发布包交接状态。
- 后续如公司指定前端技术栈，需要新增 ADR 记录替换原因和影响范围。
